PHP Webserver on Android using Termux.

This PHP web server is running directly on your Android phone using Termux! This setup allows you to carry a full-fledged web development environment in your pocket, perfect for coding and testing websites on the go.

Installation

So now, open your Termux. Before installing anything, we need to update and upgrade the system in Termux first to ensure we have the latest versions of the software.

Use the following commands below to Update and Upgrade the System on Termux.

Command

apt update
apt upgrade
apt update

Next, we need to access Termux from our Internal Storage.


Use the command below to request permission to access to your Internal Storage.

Command

termux-setup-storage

After that you will see a prompt like this ↓

Simply tap "Allow" to allow Termux from accessing your internal storage.

Next, we need to create a directory in your Internal Storage.


By running the command below. You can create a directory named "htdocs" into your phone storage.

Command

mkdir storage/shared/htdocs

To navigate to directory "storage/shared/htdocs"


Run the command below.

Command

cd storage/shared/htdocs

Next, we need to create and write a program named 'index.php' to host it with Apache.


Run the command below to enter code editor mode

Command

nano -l index.php

This is what Code Editor Looks Like ↓

Next, Write/Copy Paste the Code below ↓.

PHP

<?php
    phpinfo();

Installing Important Package/Program


Run the command below, to install necessary packages/programs

Command

apt install php-apache

Print Working Directory


Run the command below, to show the current directory

Command

pwd

Copy this path/directory later

/data/data/com.termux/files/home/storage/
shared/htdocs

Command

cd

Change your directory.


Use the command below to change your directory to target directory.

Command

cd $PREFIX/etc/apache2

Command

ls

To ensure the directory you navigate to is correct.


You need to see this lists of files

Afterthat, You need to enter in Code Editor mode.


Use the command below to enter Code Editor mode on the selected file.

Command

nano -l httpd.conf

And you are in Code Editor mode.

Now, go to line 53, and copy paste the code below.

Command

ServerName localhost

Code

LoadModule php_module libexec/apache2/libphp.so
<FilesMatch \.php$>
